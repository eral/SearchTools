# SearchTools
Unityに於ける各種検索機能の実装

## 使い方

0. Unityメニュー/Window/Search ToolsからRom Include Viewerを開く
0. Rom Include Viewerが開くので解析が終わる迄待つ
0. Projectタブから梱包状況を知りたいアセットを選択する
0. Rom Include Viewerに選択したアセットの梱包状況が表示される
